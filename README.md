Программа для проведения выравнимания пар цепочек нуклеотидов модифицированным методом Нидлмана-Вунша.

В данной программе матрица виравниваний, которая необходима для работы метода Нидлмана-Вунша, рассчитавается блоками, и процесс её расчета может быть выполнен параллельно. 
Для распараллеливания вычислений используются технологии OpenCL и OpenMP.
